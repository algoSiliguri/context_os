# ADR 0001: 5-Layer Separation

**Date:** 2026-04-15
**Status:** Accepted

## Context

The system spans Brain (Layer 1), Design (Layer 2), Execution (Layer 3),
Reality (Layer 4), Reflection (Layer 5). They could share state and call
each other freely.

## Decision

**Layer 3 NEVER touches Layer 1.** No LLM calls, no brain queries, no
network I/O during the trade path. Strategy is a pure function of market
data → signal. Risk is pure config → veto/approve. Execution is pure
approved-signal → broker call.

Reality (Layer 4) is **append-only** during execution. No updates to past
decision_log or trades records.

Reflection (Layer 5) and the Brain (Layer 1) read Layer 4 *after* sessions,
not during.

## Consequences

- Determinism: same market data → same trade output, every time
- Reproducibility: any session can be replayed from Layer 4 records
- No async surprises: no LLM latency in the trade path
- Cost: cross-layer optimizations are forbidden (acceptable trade)
